Introduction
Snake Game is a simple and fun game where the player controls a snake that moves around the game board, eating food pellets to grow longer. The game ends if the snake collides with itself or with the walls of the game board.

Features
Classic arcade-style gameplay.
Responsive design.
Score tracking.
Adjustable game speed.
Game over screen with option to restart.

How to Play
Open the game in your web browser.
Use the arrow keys on your keyboard to control the direction of the snake.
Guide the snake to eat the food pellets that appear on the game board.
The snake will grow longer each time it eats a pellet.
Avoid colliding with the walls of the game board or with the snake's own body.
Try to achieve the highest score possible before the game ends.

Controls
Arrow Keys: Control the direction of the snake.

Installation
To play the game locally, follow these steps:

Clone this repository to your local machine.
Navigate to the project directory.
Open the index.html file in your web browser.
Alternatively, you can play the game online here.
